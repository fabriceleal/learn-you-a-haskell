# Learn you a haskell
http://learnyouahaskell.com/

## On ghci

### Load file

```
ghci> :l file
```

*Don't need to add the .hs extension*

### Reload loaded files

```
ghci> :r
```

### Check the type of something

```
ghci> :t <something>
```

### Quit from ghci

```
ghci> :quit
```

## Import module

```
ghci> :m + <module.name>
```

Like this:

```
ghci> :m + Data.List
```
